# To-Do List em Java 

Projeto desenvolvido para estudar lógica, orientação a objetos e manipulação de arquivos em Java.

## Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Marcar como concluída
- Remover tarefas
- Persistência de dados (Salva em .txt automaticamente)

## Como rodar
Compile e execute a partir da pasta raiz:
javac To_Do_List_java/*.java
java To_Do_List_java.Main
